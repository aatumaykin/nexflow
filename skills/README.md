# Навыки (Skills)

В этой директории находятся модульные навыки/компоненты для MCP (Model Context Protocol) интеграций.

## Структура

```
skills/
  └── <skill-name>/
      ├── <skill>.go
      ├── <skill>_test.go
      └── README.md
```

## Назначение

- Реализация MCP skills
- Интеграции с внешними системами
- Специфические компоненты для AI агентов
