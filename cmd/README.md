# Команды проекта

В этой директории находятся точки входа для приложений (main packages).

Каждое приложение (сервис, CLI инструмент и т.д.) должно иметь свою поддиректорию в `cmd/`.

## Структура

```
cmd/
  └── <app-name>/
      └── main.go
```

## Пример

Для создания нового сервиса:

```bash
mkdir -p cmd/<service-name>
touch cmd/<service-name>/main.go
```
