# Внутренний код проекта

В этой директории находится код, предназначенный только для использования внутри этого проекта.

Код в `internal/` не может быть импортирован другими модулями.

## Структура

```
internal/
  ├── <module>/
  │   ├── domain/       # Сущности бизнес-домена
  │   ├── repository/   # Репозитории (доступ к данным)
  │   ├── service/      # Бизнес-логика
  │   └── handler/      # Обработчики (HTTP/gRPC endpoints)
```
