# Переиспользуемые пакеты

В этой директории находится код, который может быть использован другими проектами.

Пакеты в `pkg/` должны быть независимыми и иметь стабильный API.

## Структура

```
pkg/
  ├── <package-name>/
  │   ├── <file>.go
  │   └── <file>_test.go
```

## Правила

- Пакеты должны быть независимыми и иметь минимум зависимостей
- API пакетов должен быть стабильным
- Каждому пакету нужны тесты
